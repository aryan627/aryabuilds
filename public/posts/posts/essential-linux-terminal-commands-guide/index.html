Essential Linux Terminal Commands Guide

2024-12-26
#linux 

Navigation Commands#

pwd (Print Working Directory)#

Shows your current location in the file system.

pwd
/home/user/documents

ls (List)#

Lists files and directories in the current location.

# Basic usage
ls

# Show hidden files
ls -a

# Show detailed information
ls -l

# Combine options
ls -la

cd (Change Directory)#

Move between directories.

# Go to home directory
cd

# Go up one level
cd ..

# Go to specific directory
cd /path/to/directory

# Go to previous directory
cd -

File Operations#

mkdir (Make Directory)#

Create new directories.

# Create single directory
mkdir new_folder

# Create nested directories
mkdir -p parent/child/grandchild

touch#

Create empty files or update file timestamps.

# Create new file
touch newfile.txt

# Create multiple files
touch file1.txt file2.txt file3.txt

cp (Copy)#

Copy files and directories.

# Copy file
cp source.txt destination.txt

# Copy directory and its contents
cp -r source_folder destination_folder

mv (Move)#

Move or rename files and directories.

# Move file
mv file.txt /new/location/

# Rename file
mv oldname.txt newname.txt

rm (Remove)#

Delete files and directories.

# Remove file
rm file.txt

# Remove directory and contents
rm -r directory/

# Force remove (use with caution!)
rm -f file.txt

File Content Operations#

cat (Concatenate)#

Display file contents or combine files.

# View file contents
cat file.txt

# Combine files
cat file1.txt file2.txt > combined.txt

less#

View file contents page by page.

less largefile.txt
# Use space to move forward
# Use 'b' to move backward
# Use 'q' to quit

head#

View the beginning of a file.

# Show first 10 lines
head file.txt

# Show first n lines
head -n 5 file.txt

tail#

View the end of a file.

# Show last 10 lines
tail file.txt

# Show last n lines
tail -n 5 file.txt

# Follow file updates
tail -f log.txt

System Information#

top#

Monitor system processes and resource usage.

top
# Press 'q' to quit

df (Disk Free)#

Show disk space usage.

# Show human-readable sizes
df -h

du (Disk Usage)#

Show directory space usage.

# Show directory size
du -h directory/

# Show total size only
du -sh directory/

File Permissions#

chmod (Change Mode)#

Modify file permissions.

# Make file executable
chmod +x script.sh

# Set specific permissions
chmod 644 file.txt

chown (Change Owner)#

Change file owner and group.

# Change owner
chown user file.txt

# Change owner and group
chown user:group file.txt

Text Processing#

grep (Global Regular Expression Print)#

Search for patterns in files.

# Basic search
grep "pattern" file.txt

# Case-insensitive search
grep -i "pattern" file.txt

# Recursive search in directory
grep -r "pattern" directory/

wc (Word Count)#

Count lines, words, and characters.

# Count lines, words, and characters
wc file.txt

# Count only lines
wc -l file.txt

Tips and Tricks#

Tab Completion#

  • Press Tab once to auto-complete commands or file names
  • Press Tab twice to show all possible completions

Keyboard Shortcuts#

  • Ctrl + C: Cancel current command
  • Ctrl + L: Clear screen
  • Ctrl + A: Move to beginning of line
  • Ctrl + E: Move to end of line
  • Ctrl + U: Clear line before cursor

Getting Help#

man (Manual)#

Access command documentation.

man command_name

help#

Get information about built-in shell commands.

help command_name

whatis#

Get a brief description of a command.

whatis command_name

Note: This guide covers basic usage of common Linux commands. Many commands have additional options and features accessible through their man pages or help documentation.
